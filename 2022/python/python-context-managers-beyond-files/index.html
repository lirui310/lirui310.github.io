上下文管理器的工作原理

上下文管理器通过实现__enter__和__exit__方法来工作：

  • __enter__：在进入with语句块之前调用，返回值会被赋给as后面的变量
  • __exit__：在离开with语句块时调用，无论是正常退出还是发生异常
